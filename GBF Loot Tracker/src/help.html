Granblue Fantasy Loot Tracker Module v1.0.4

Frequently Asked Questions
• Is this extension safe?

Yes, you shouldn't be detected using this extension. This extension uses chrome's debugger library to listen to the network traffic of any tab on the results screen of a raid. Another popular extension “Cypher” uses the same technique, and no one has been banned yet. I would still advise using this extension at your own risk.

• How does this extension work?

The extension catches a file containing the results of any battle you complete, processes the data, and then saves it. You can read more about exactly how the extension does that HERE.

• I keep seeing this annoying message, is there any way to remove it?

Yes, to stop seeing the message, you can enable silent debugging. To do this, right click on the Google Chrome shortcut on your desktop. Select “Shortcut” and click on the “Target:” field. Paste the flag --silent-debugger-extension-api at the end of the file path. On my system, the full field is "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --silent-debugger-extension-api. Depending on where chrome is installed, the path could be different.

• One of the pictures for loot is showing up as the April fools item vyrnball, why is this?

This will happen if you receive a drop that didn't have it's image saved in the extension. This image will replace any image that you are currently missing. To fix it, place the missing image in the /src/img/loot folder and name the image that item's id. This should fix the issue.

• I have encountered a bug. Where can I report it?

This extension is still new and under development so there is probably still a few bugs. Please send me an email at granbluetracker@gmail.com telling me what happened, and I'll try to fix it.

• A raid that I completed was not tracked. What happened?

Because of how the extension works, it is not able to see what enemy you killed. The way that the extension determines the boss you killed is by comparing the loot it dropped to what loot each tracked raid drops. If multiple raids can drop the exact loot you received, there is no way to tell which of those raids you did. Certain raids are also not recorded in the tracker, (such as magna 1).

• How do I change the boss I am trying to track?

On the “loot module”, click on the gear on the top right to open another panel. Here you can change the current boss, as well as the period you want to view data from.

• How do I export or import my raid data?

Currently with this version, you are not able to do that. With a future version, you should be able to view, import, and export any raid data.

• How do I exclude certain items from being tracked in my raids?

This feature has not been implemented yet and should come with a future update.

• How does the “Timer” option under the “Data Period” setting work?

Selecting this will start a timer and show all loot collected after the timer was started. Changing the “Data Period” setting to another period and back to “Timer” will reset it.
